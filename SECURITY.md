# Security Policy

## Supported versions

Only the latest published NuGet package version is actively supported for security fixes.

## Reporting a vulnerability

Please do not open a public issue for security vulnerabilities.

Report vulnerabilities privately through GitHub Security Advisories for this repository.

If private advisories are unavailable, contact maintainers through repository owner channels and include "SECURITY" in the subject.

Include:
- A clear description of the issue
- Reproduction steps or proof of concept
- Potential impact
- Suggested mitigation if available

We will acknowledge receipt within 3 business days and provide status updates as remediation progresses.
